## ğŸ¯ Best Practices

### Images:
- Max 2-3 images per email
- Use WebP for best compression
- Alt text for all images
- Logo: 200-300px width

- Headers: 24-36px (H1), 20-24px (H2)

- Line height: 1.5-1.6
- Max 2-3 fonts

### Layout:
- Max width: 600px
- Mobile-first design
- Clear visual hierarchy
- Generous whitespace

### Content:
- Short paragraphs (2-3 sentences)
- One primary CTA
- Personalize with variables
- Test before sending
